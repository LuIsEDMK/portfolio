# Portfólio - Luis EDMK

Portfólio profissional de Data Scientist & ML Engineer.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Tailwind CSS (via CDN)
- Font Awesome

## 📁 Estrutura

```
portfolio/
├── index.html          # Página principal
├── css/
│   ├── style.css      # Estilos principais
│   └── animations.css # Animações
├── js/
│   ├── data.js        # Dados (projetos, skills)
│   ├── main.js        # Funcionalidades principais
│   ├── typing.js      # Efeito de digitação
│   └── animations.js  # Animações de scroll
└── assets/
    └── profile.jpg    # Sua foto de perfil
```

## 🚀 Como usar

### 1. Adicione sua foto de perfil

Coloque sua foto em `assets/profile.jpg`

### 2. Personalize os dados

Edite `js/data.js` para alterar:
- Projetos
- Habilidades
- Textos do efeito de digitação

### 3. Visualize

Abra `index.html` no navegador ou use um servidor local:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```
